# Jira Bug 이슈 생성

Bug 유형의 Jira 이슈를 생성합니다.

## 필수 정보 수집

1. **프로젝트 키** - Jira 프로젝트 키
2. **제목** - 버그 제목 (간결하게)
3. **현상** - 무엇이 잘못되었는가?
4. **재현 방법** - 버그 재현 단계
5. **기대 동작** - 정상적으로 동작해야 하는 방식
6. **원인 분석** - 알고 있다면 (선택)

## 템플릿 형식

```
## 현상
{무엇이 잘못되었는가}

## 재현 방법
1. {단계 1}
2. {단계 2}
3. {단계 3}

## 기대 동작
{정상적으로 동작해야 하는 방식}

## 원인 분석
{알고 있다면 작성, 없으면 "분석 필요"}

## 참고자료
- {스크린샷, 로그, 에러 메시지 등}
```

## 실행 절차

1. 필수 정보가 없으면 AskUserQuestion으로 확인
2. 템플릿 형식으로 description 구성
3. `mcp__jira__jira_create_issue` 호출 (issue_type: "Bug")
4. 생성된 이슈 키 반환
