# Jira Task 이슈 생성

Task 유형의 Jira 이슈를 생성합니다.

## 필수 정보 수집

사용자에게 다음 정보를 확인:

1. **프로젝트 키** - Jira 프로젝트 키 (예: GIFCA, DEV)
2. **제목** - 이슈 제목 (간결하게)
3. **작업배경** - 왜 이 작업이 필요한가?
4. **작업내용** - 구체적으로 무엇을 해야하는가?
5. **완료조건** - 작업 완료 기준
6. **참고자료** - 관련 문서, 링크 (선택)

## 템플릿 형식

```
## 작업배경
왜 이 작업이 필요한가?
- [x] {기술부채/성능개선/요청사항/기타}

{배경 상세 설명}

## 작업내용
1. {구체적 작업 1}
2. {구체적 작업 2}
3. {구체적 작업 3}

## 완료조건
- [ ] {조건 1}
- [ ] {조건 2}
- [ ] 테스트 통과
- [ ] 코드 리뷰 완료

## 참고자료
- {링크 또는 문서}
```

## 실행 절차

1. 필수 정보가 없으면 AskUserQuestion으로 확인
2. 템플릿 형식으로 description 구성
3. `mcp__jira__jira_create_issue` 호출
4. 생성된 이슈 키 반환

## 예시

```
/jira-task
> 프로젝트: GIFCA
> 제목: 바우처 만료 알림 배치 구현
> 배경: 사용자가 바우처 만료를 인지하지 못해 불만 증가
> 작업: 만료 3일 전 푸시 알림 발송 배치 구현
> 완료조건: 배치 정상 실행, 알림 발송 확인
```

→ GIFCA-123 이슈 생성됨
